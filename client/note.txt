REQUIREMENT

-npm install @mui/material @emotion/react @emotion/styled
-npm install @mui/icons-material
-npm install jwt-decode
-npm install uuid
-npm install firebase
-npm install react-dropzone  ====> HTML5 uyumlu bir sürükle ve bırak bölgesi oluşturmak için basit React kancası.
-npm i mapbox-gl => web üzerinde etkileşimli, özelleştirilebilir vektör haritaları için bir JavaScript kitaplığıdır. 
-npm i react-map-gl => Mapbox GL JS uyumlu kitaplıklar için bir React API sağlamak üzere tasarlanmış bir React bileşenleri paketidir 
-npm i @mapbox/mapbox-gl-geocoder => Mapbox Geocoding API'sini kullanan mapbox-gl-js için bir coğrafi kodlayıcı kontrolü .
